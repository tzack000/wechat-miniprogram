# Change: 增加课程预约功能（支持选择教练）

## Why

用户除了预约场馆外，还希望能够预约专业课程（如瑜伽课、游泳培训、篮球训练等），并在预约时选择自己喜欢的教练。这将提升用户体验，增加平台的服务价值。

## What Changes

### 新增功能

- **教练管理**:
  - 教练信息展示（姓名、头像、简介、擅长领域、评分）
  - 教练列表浏览和筛选
  - 教练详情页面

- **课程管理**:
  - 课程信息（名称、类型、描述、时长、价格、人数上限）
  - 课程与教练关联
  - 课程与场馆关联（可选）

- **课程预约**:
  - 浏览课程列表（按类型、教练筛选）
  - 查看课程详情和排期
  - 选择教练预约课程
  - 查看剩余名额
  - 预约课程

- **用户功能**:
  - 我的课程预约列表
  - 取消课程预约
  - 课程签到（可选）

- **管理员功能**:
  - 教练管理（增删改查）
  - 课程管理（增删改查）
  - 课程排期管理
  - 课程预约管理

## Impact

- Affected specs:
  - `specs/course-booking/spec.md` (新增)
- Affected code:
  - 新增数据库集合：`coaches`、`courses`、`course_schedules`、`course_bookings`
  - 新增云函数：`coach`、`course`
  - 新增页面：课程列表、课程详情、教练列表、教练详情、我的课程
  - 修改页面：管理后台（新增课程管理、教练管理入口）
- Dependencies: 
  - 依赖现有用户认证模块
  - 可复用场馆预约的 UI 组件和交互模式
