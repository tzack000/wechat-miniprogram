# ⚡ 快速部署指南

> 3分钟完成新云函数部署和测试数据初始化

## 🎯 部署步骤（仅3步）

### 步骤1️⃣：部署云函数（30秒）

在微信开发者工具中：

```
1. 找到左侧 cloudfunctions 文件夹
2. 右键点击 initAllTestData 文件夹
3. 选择「上传并部署：云端安装依赖」
4. 等待出现"部署成功"提示
```

**成功标志：** 文件夹图标变为 ☁️，名称前有 🟢

---

### 步骤2️⃣：打开云开发控制台（10秒）

```
1. 点击工具栏的「云开发」按钮
2. 在左侧菜单选择「云函数」
3. 找到并点击 initAllTestData
```

---

### 步骤3️⃣：初始化测试数据（1分钟）

在云函数详情页：

```
1. 点击右上角「测试」按钮
2. 输入以下JSON（复制粘贴）：
```

```json
{
  "action": "all",
  "clear": true
}
```

```
3. 点击「运行测试」
4. 等待5-10秒
5. 看到 "success": true 即成功
```

---

## ⚠️ 遇到问题？

### 部署失败？
→ 查看控制台错误信息  
→ 确认已开通云开发环境  
→ 尝试刷新后重新部署

### 找不到云函数？
→ 刷新云函数列表  
→ 确认部署成功（看到✓）  
→ 等待10秒后再查看

### 测试超时？
→ 分批初始化（先初始化 users 和 venues）  
→ 使用以下参数：
```json
{
  "modules": ["users", "venues"],
  "clear": true
}
```
